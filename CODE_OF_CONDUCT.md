# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its community code of conduct.

## Scope

The Contributor Covenant applies to all project spaces — GitHub issues and pull requests, repository discussions, and any official communication channels operated under the AgenticZK name.

## Reporting

Instances of behavior that violate the Contributor Covenant may be reported to:

- **Email:** `0x@arcent.ink`
- **GitHub:** Private Vulnerability Reporting (for serious safety concerns) — see [`SECURITY.md`](SECURITY.md)

All reports are reviewed promptly and confidentially. The project maintainer is committed to enforcing the Contributor Covenant fairly and consistently.

## Full Text

The complete Code of Conduct is maintained upstream by the Contributor Covenant project:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

We do not vendor the full text into this repository to ensure contributors always read the canonical, up-to-date version.

## Versioning

This project follows Contributor Covenant **2.1**. Any future migration to a newer Contributor Covenant version will be announced in [`CHANGELOG.md`](CHANGELOG.md) before taking effect.
